# 🎯 Trackify

> Your personal productivity and fitness tracking companion powered by AI

Trackify is a comprehensive productivity and fitness tracking application that helps you set goals, track daily tasks, monitor workouts, and maintain consistent habits—all enhanced with AI-powered motivational coaching and a gamified XP system.

🌐 **Live Demo:** [trackify-v1.vercel.app](https://trackify-v1.vercel.app/)

![Trackify Dashboard](https://via.placeholder.com/800x400/89CFF0/ffffff?text=Trackify+Dashboard)

## ✨ Features

### 🤖 AI-Powered Motivation Coach
- Personalized motivational messages generated by Google Gemini AI
- Context-aware encouragement based on your progress and consistency
- Refreshable motivation to keep you inspired

### 🎮 Gamification System
- **XP Rewards**: Earn points for completing tasks and workouts
- **Streak Bonuses**: Multi-day streak rewards for consistency
- **Level Progression**: Watch your level grow as you achieve more
- **Achievement Collection**: Completed goals become trophies

### 📊 Comprehensive Tracking
- **Goals Management**: Set and track long-term objectives
- **Daily Tasks**: Break down goals into actionable daily items
- **Workout Logging**: Record and track fitness activities
- **Analytics Dashboard**: Visualize your progress with interactive charts

### 🎨 Modern UI/UX
- Clean, responsive design with light/dark mode support
- Smooth animations and transitions
- Accessible components built with Radix UI
- Mobile-first approach

## 📦 Installation

### Prerequisites
- Node.js 20+ and npm
- Firebase account
- Google AI API key (for Gemini)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/amolwaghmare05/trackify.git
   cd trackify
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Firebase**
   
   Create a `.env.local` file in the root directory:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
   GOOGLE_GENAI_API_KEY=your_gemini_api_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open [http://localhost:3000](http://localhost:3000)** in your browser
